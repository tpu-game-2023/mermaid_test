# 課題項目
## 流れ図
### 条件
- 開始と終了ノードをつける
- 条件分岐を組み込む
- 5ノード以上
- カッコいいほど高得点

## 解答
```mermaid
flowchart LR;
  A([開始]) --> B[/手を洗う/];
  B --> C{レシピを決めたか};
  C -- 真 --> D[料理はじめ！];
  C -- 偽 --> E[夕飯抜き！];

```

