# 課題項目
## 流れ図
### 解答
- 五目並べの判定文
```mermaid
flowchart LR;
  Z([開始]) --> A;
  A[/設置/] --上からチェック--> B{マスを確認};
  B -- 真 --> C[/個数カウント増加/];
  B -- 偽 --> D{方向チェック};
  C --> E{個数カウントチェック};
  E -- 1, 2, 3, 4, --> F{一つ先をチェック};
  F --真--> C;
  F --偽--> G[個数カウントリセット];
  G --> D;
  D --右上以外--> I[時計回りで方向切り替え];
  I --> B;
  D -- 右上 --> J(相手ターンへ);
  J --> A;
  E -- 5 --> H{勝敗判定};
  subgraph "勝敗"
   H --自分の勝ち-->K[勝利];
   H --相手の勝ち-->L[敗北];
   K --> M([終了]);
   L --> M;
  end
```
